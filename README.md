# Número Secreto

## Sobre o projeto

Este repositório não tem como objetivo demonstrar minhas habilidades de programação ou apresentar um projeto desenvolvido integralmente por mim.

O código-base utilizado neste projeto foi disponibilizado como exemplo pela Alura durante o curso de Git e GitHub.

## Objetivo

O objetivo deste repositório foi servir como ambiente de prática para o aprendizado e desenvolvimento de habilidades relacionadas ao Git e ao GitHub, incluindo:

- Criação e gerenciamento de repositórios
- Commits e histórico de versões
- Branches
- Merge
- Revert
- Reset
- Resolução de conflitos
- Versionamento de código

## Observação

Embora o código original tenha sido fornecido pela Alura, todas as operações de versionamento registradas neste repositório foram realizadas por mim durante os estudos e exercícios práticos do curso.

Este repositório é mantido como registro da minha evolução e aprendizado em Git e GitHub.

## Créditos

Código-base disponibilizado pela Alura para fins educacionais.

Curso: Git e GitHub
Plataforma: Alura
